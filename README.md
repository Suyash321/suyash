# Personal Portfolio – Suyash Tiwari

Modern single-page portfolio built with **React + TypeScript + Vite**, showcasing my front-end development and UI/UX work.

## Live site

- URL: https://suyashtiwari-five.vercel.app/

## Tech stack

- React with TypeScript
- Vite build tool
- Modern CSS (responsive layout, animations)
- Deployed on Vercel

## Features

- Hero section with introduction and primary call-to-action
- Projects section highlighting selected work with descriptions and links
- Skills and tools section
- About section with background and focus areas
- Contact section with direct links (email, social profiles)

## Getting started

```bash
# clone the repo
git clone https://github.com/Suyash321/suyash.git
cd suyash

# install dependencies
npm install

# start dev server
npm run dev

# build for production
npm run build
